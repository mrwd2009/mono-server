# Overview
This project is used to provide api and some common services.
Now it's a basic service structure, other interesting things will be added in future.

Node version >= 14.15.1

## Available Scripts

### `yarn install`
Install all dependencies.

### `yarn dev`

Runs the app in the development mode.<br />
Service point is [http://localhost:4100](http://localhost:4100).

### `yarn build`

Compile source ts files.

### `Other`

I have created some other services like log and queue, you can view them in `pm2.config.js`.
